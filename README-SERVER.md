# كيفية تشغيل السيرفر

## الطريقة 1: استخدام ملف Batch (الأسهل)
1. انقر نقراً مزدوجاً على `start-server.bat`
2. سيتم تشغيل السيرفر تلقائياً
3. افتح المتصفح واذهب إلى: `http://localhost:8000/index.html`

## الطريقة 2: من Terminal
```bash
cd "D:\مشروع تخرج"
python -m http.server 8000
```

## الطريقة 3: من PowerShell
```powershell
Set-Location "D:\مشروع تخرج"
python -m http.server 8000
```

## بعد تشغيل السيرفر:
- افتح المتصفح
- اذهب إلى: `http://localhost:8000/index.html`
- أو: `http://127.0.0.1:8000/index.html`

## ملاحظة مهمة:
- تأكد من أنك في مجلد المشروع قبل تشغيل السيرفر
- إذا ظهرت قائمة مجلدات، فأنت في المجلد الخاطئ
- استخدم `start-server.bat` لتجنب هذه المشكلة
