# إنـقـاذ حــيـاة - منصة التبرع بالدم

## نظرة عامة
منصة إنسانية تربط بين المحتاجين للدم والمتبرعين بطريقة آمنة ومريحة. توفر المنصة نظام رسائل داخلي لحماية خصوصية المستخدمين خاصة النساء.

## المميزات الرئيسية

### 1. التسجيل كمتبرع
- تسجيل المعلومات الأساسية (الاسم، العمر، فصيلة الدم)
- تحديد الموقع (المدينة والمنطقة)
- رقم الهاتف اختياري
- إعدادات الخصوصية لإخفاء/إظهار الرقم

### 2. البحث عن المتبرعين
- فلترة حسب فصيلة الدم
- البحث حسب المدينة والمنطقة
- عرض معلومات المتبرعين المتطابقين

### 3. نظام الرسائل الداخلي (حل مشكلة الخصوصية)
- **المشكلة**: حماية أرقام الهواتف من الإزعاج خاصة للنساء
- **الحل**: نظام رسائل داخلي في المنصة
  - المستفيد يرسل رسالة داخل المنصة
  - يمكن اختيار مشاركة الرقم في الرسالة أو عدم مشاركته
  - المتبرع يرى الرسالة في صندوق الوارد
  - لا يتم إظهار الأرقام مباشرة في قائمة المتبرعين (إلا إذا اختار المتبرع ذلك)
  - نظام إشعارات للرسائل الجديدة

### 4. الملف الشخصي
- عرض جميع المعلومات
- تعديل إعدادات الخصوصية
- تسجيل الخروج

## التقنيات المستخدمة
- HTML5
- CSS3 (مع تصميم متجاوب)
- JavaScript (Vanilla JS)
- LocalStorage لتخزين البيانات

## كيفية الاستخدام

### 1. فتح المشروع
افتح ملف `index.html` في المتصفح

### 2. التسجيل
- اضغط على "التسجيل" في القائمة
- املأ جميع الحقول المطلوبة
- اختر فصيلة دمك
- حدد المدينة والمنطقة
- (اختياري) أدخل رقم الهاتف
- اختر إذا كنت تريد إظهار رقمك أم لا

### 3. البحث عن متبرع
- اضغط على "البحث عن متبرع"
- اختر فصيلة الدم المطلوبة
- (اختياري) حدد المدينة والمنطقة
- اضغط "بحث"
- اضغط "إرسال رسالة" للتواصل مع المتبرع

### 4. إرسال رسالة
- بعد الضغط على "إرسال رسالة"
- اكتب رسالتك
- (اختياري) أدخل رقمك للتواصل
- اضغط "إرسال"

### 5. قراءة الرسائل
- اضغط على "الرسائل" في القائمة
- ستظهر جميع الرسائل الواردة والصادرة
- الرسائل الجديدة تظهر بشكل مميز

## حل مشكلة الخصوصية

### المشكلة الأصلية
- إظهار أرقام الهواتف مباشرة قد يسبب إزعاج خاصة للنساء
- عدم وجود حماية من المكالمات غير المرغوب فيها

### الحل المطبق
1. **إعدادات الخصوصية**: كل متبرع يختار إذا كان يريد إظهار رقمه أم لا
2. **نظام الرسائل الداخلي**: 
   - التواصل يتم عبر رسائل داخل المنصة
   - المستفيد يختار إذا كان يريد مشاركة رقمه في الرسالة
   - المتبرع يقرر الرد أو عدم الرد
3. **عدم إظهار الأرقام مباشرة**: 
   - إذا اختار المتبرع إخفاء رقمه، يظهر "الرقم مخفي - استخدم الرسائل للتواصل"
   - المستفيد يضطر لاستخدام نظام الرسائل

### ميزات إضافية لحماية الخصوصية
- يمكن للمتبرع تغيير إعدادات الخصوصية في أي وقت
- الرسائل محفوظة في المنصة فقط
- لا يتم مشاركة الأرقام إلا بموافقة صريحة

- **التصميم**: حديث ونظيف

2. نظام مصادقة وتأمين
3. إشعارات بريد إلكتروني
4. تطبيق موبايل
5. خريطة تفاعلية للمتبرعين
6. نظام تقييم للمتبرعين
